Data Analytics

Subject project Data Analytics

Project Overview

The project implements data scraping and data visualization. The main research data is agricultural data (seasonal rice) in Vietnam from 1995 to 2020.
All data are obtained from the website of the General Statistics Office of Vietnam: https://www.gso.gov.vn/
We conduct data visualization in 3D environment to get a more general view of the data. The correlation of the data columns is also evident. The program is then built on Oculus glasses for easy viewing and interaction.
In addition to analysis, our system also predicts output in the next years (based on existing data). Details of the presentation file can be found here
